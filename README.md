Vagrant Builds
====================

Baseline Wordpress build using Vagrant and Starkers

Feel free to fork/clone/use it if you want to.


Included versions
---------------------

### Wordpress
- Wordpress v4.0
- Starkers v4.0

### Libraries
- jQuery v1.11.0
- HammerJS v1.0.8
- iOS Orientation Change Fix (authored by @scottjehl, rebounded by @wilto)
